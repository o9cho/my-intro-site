요약

  • Unity–Node.js–MongoDB 연동을 통해 랭킹 시스템의 전체 구조를 직접 설계 및 구현
  • 클라이언트–서버–DB 간 데이터 흐름을 이해하고, 실제 동기화 로직으로 검증

팀

  • 프로젝트 진행기간 : 2020.07
  • 팀 : 개인
  • 담당 파트 : 프로그래밍(클라이언트, 백엔드)

주요 내용

  • Unity 클라이언트와 Node.js 기반 서버 간 통신 구조 설계 및 구현
  • MongoDB를 활용한 유저 점수 저장·정렬 로직 개발
  • 비동기 데이터 요청/응답 구조 실습 및 예외 처리 설계
  • 단순 연동이 아닌, 실시간 데이터 갱신 및 랭킹 표시 구조까지 구현

성과

  • 클라이언트와 서버 간 데이터 흐름에 대한 실질적 이해 확보
  • Node.js + MongoDB + Unity 통합 개발 경험으로 게임 서버 구조 설계 역량 강화
